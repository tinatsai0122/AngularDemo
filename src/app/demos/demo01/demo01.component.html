Demostration 01 - (Strings) Interpolation de chaines (One-way binding)

L'interpolation, notee avec des doubles accolades {{ ... }}, permet d'inserer dynamiquement des valeurs variables dans le contenu HTML.

Recuperation des valeurs se fait de TypeScript vers HTML.

Affichages des valeurs:

Type primitif : string {{ variable1 }}

Type primitif : number {{ variable2 }}

Type primitif : boolean {{ variable3 }}

Array : string[] {{ variable4 }}

Array : number[] {{ variable5 }}

Array : boolean[] {{ variable6 }}

Definite Assignment Assertion : string {{ variable7 }}

Ambient Declaration : number {{ variable8 }}

Any: any[] {{ variable9[0] }}

Any: any[] {{ variable9[1] }}

Any: any[] {{ variable9[2]() }}

Methods: function anonyme () {{ variable10() }}

Methods: function arrow () {{ variable11() }}

Methods: a:5 et b:3 () {{ variable12(5, 3) }}

Methods: a:5 et b:3 () {{ variable13(5, 3) }}
