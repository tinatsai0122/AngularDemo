Demostration 06 - Les Pipes

Les pipes sont des fonctions qui permettent de transformer une expression dans la vue.

Pour utiliser une pipe, vous devez ecrire {{ expression | nomPipe }}

Exemples :

Pipe String

uppercase : {{ "Bonjour tout le monde. Je m'appelle Tina." | uppercase }}
lowercase : {{ "Bonjour tout le monde. Je m'appelle Tina." | lowercase }}
titlecase : {{ "Bonjour tout le monde. Je m'appelle Tina." | titlecase }}
slice: 0:5 : {{ "Bonjour tout le monde. Je m'appelle Tina." | slice : 0 : 5 }}
json : {{ "Bonjour tout le monde. Je m'appelle Tina." | json }}

Decimal Pipes | Number Pipe

number: '1.2-2' : {{ 3.14159 | number : "1.2-2" }}
percent : {{ 0.14159 | percent }}
currency: 'EUR' : {{ 14159 | currency : "EUR" }}

Avant:{{ 1234.5678 }}

Apres:{{ 1234.5678 | number }}

Apres:{{ 1234.5678 | number : "5.2-5" }}

Apres:{{ 1234.5678 | number : "5.2-5" : "fr-BE" }}
